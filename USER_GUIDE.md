# User Guide - Collaborative White Board

Welcome to the Collaborative White Board! This guide will help you understand all the features and how to use them effectively.

## Getting Started

When you launch the application, you'll see:
- A large white canvas in the center (the drawing area)
- A toolbar panel on the right side with various controls

## Features Overview

### 1. Drawing on the Canvas

**How to Draw:**
1. Move your mouse cursor over the white canvas
2. Click and hold the left mouse button
3. Move the mouse while holding the button to draw
4. Release the button to stop drawing

**Tips:**
- The drawing follows your mouse movements smoothly
- You can draw continuously by keeping the mouse button pressed
- Lift the mouse button to create separate strokes

### 2. Color Selection

The **Colors** section provides 5 color options:

- **Black**: Default drawing color
- **Red**: For highlights or important annotations
- **Blue**: Cool tones for diagrams
- **Green**: Natural tones
- **Yellow**: Bright highlights (note: may be less visible on white background)

**How to Change Colors:**
1. Click on any of the colored square buttons
2. The "Current Color" indicator will update to show your selection
3. All new drawings will use this color

### 3. Brush Thickness

The **Thickness** section offers three pen sizes:

- **Thin (2px)**: For fine details and precise work
- **Medium (5px)**: Balanced option for general use
- **Thick (10px)**: For bold strokes and filling areas

**How to Change Thickness:**
1. Click on one of the thickness buttons
2. The "Current" indicator will show your selected thickness
3. New drawings will use this thickness

### 4. Actions

#### Clear Board
- **Purpose**: Removes all drawings from the canvas
- **How to Use**:
  1. Click the "Clear Board" button (red background)
  2. A confirmation dialog will appear
  3. Click "Yes" to clear or "No" to cancel

#### Save Drawing
- **Purpose**: Saves your current drawing as an image file
- **Supported Formats**: PNG, JPEG, BMP
- **How to Use**:
  1. Click the "Save Drawing" button (green background)
  2. Choose a location and filename
  3. Select your preferred image format
  4. Click "Save"

**Tips:**
- PNG format is recommended for best quality
- Files are automatically named with timestamp by default
- You can save multiple versions of your work

#### Load Drawing
- **Purpose**: Opens a previously saved image onto the canvas
- **Supported Formats**: PNG, JPEG, BMP, and other image formats
- **How to Use**:
  1. Click the "Load Drawing" button (blue background)
  2. Navigate to your image file
  3. Select the file and click "Open"
  4. The image will be loaded onto the canvas

**Notes:**
- Loading an image will clear any existing drawings
- Images are scaled to fit the canvas size
- You can continue drawing on top of loaded images

## Keyboard Shortcuts

Currently, the application uses mouse-based controls. Future versions may include keyboard shortcuts.

## Window Management

### Resizing the Window
- The application window can be resized
- The canvas automatically adjusts to the new size
- Minimum window size: 800x600 pixels
- The toolbar panel remains fixed width on the right

### Window Position
- The application starts centered on your screen
- You can drag it to any position on your monitor

## Tips and Best Practices

1. **Save Frequently**: Get in the habit of saving your work regularly
2. **Use Different Colors**: Color-code your notes or diagrams for better organization
3. **Experiment with Thickness**: Different thicknesses work better for different tasks
4. **Plan Before Drawing**: Think about your layout before starting complex diagrams
5. **Start Fresh**: Don't hesitate to clear and start over if needed

## Common Use Cases

### 1. Quick Sketches
- Use thin or medium thickness
- Stick with black for simple sketches
- Save when satisfied

### 2. Annotations
- Load an existing image
- Use red or blue for highlights
- Use medium or thick thickness for visibility

### 3. Diagrams and Flowcharts
- Use different colors for different elements
- Use thin thickness for connecting lines
- Use medium thickness for boxes and text

### 4. Teaching/Presentations
- Use thick thickness for better visibility
- Use high-contrast colors (black, red, blue)
- Save progress for future sessions

## Troubleshooting

**Problem**: Can't see yellow drawings
- **Solution**: Yellow is difficult to see on white. Use darker colors for better visibility.

**Problem**: Drawing is jerky or slow
- **Solution**: Close other applications to free up system resources.

**Problem**: Accidentally cleared the board
- **Solution**: Unfortunately, this can't be undone. Save frequently to prevent data loss.

**Problem**: Loaded image looks distorted
- **Solution**: The image is scaled to fit. Try resizing the window before loading.
